About the website

What is Home Fixer?

Home Fixer is a web system for managing home repair services. It includes a customer interface, service request form, backend API, MongoDB database, and admin dashboard for tracking and updating repair requests.

🌐

Website + Mobile-first app

The landing page explains the service, while the app screens provide the user flow.

🧾

Repair requests

Customers submit service requests with city, address, service type and priority.

🗄️

Backend database

Requests are saved in MongoDB and managed through a Node.js API.

Services

Services available in the system

Each service can be selected by the customer when creating a repair request.

🔧

Plumbing

Leaks, faucets, pipes and kitchen or bathroom water problems.

 ⚡

Electric

Lighting, switches, sockets and home electrical issues.

 🧹

Cleaning

Deep cleaning, home cleaning and post-repair cleaning.

 🏠
